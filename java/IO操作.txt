IO流

	IO，Input/Output
	完成输入/输出
	
	应用程序运行时――――数据在内存中				――――   把数据写入硬盘（磁带）
				内存中的数据不可持久化保存的
				
	输入：	从外部存储器（硬盘、磁带、U盘）把数据读入内存。
	输出：	从内存中把数据写入外部存储器中，
			这样就可保证：即使程序退出了，数据依然不会丢失。
			
File ―― 代表磁盘上的文件或目录。

	listRoots：列出磁盘上所有的根目录
	exists：判断是否存在。
	mkdir：创建目录。
	listFiles():列出当前目录下所有的文件和子目录。
	listFiles(FileFilter filter)：列出当前目录下符合指定条件的文件和子目录。
	listFiles(FilenameFilter filter)：列出当前目录下符合指定条件的文件和子目录。
	
	列出某个磁盘所有的文件 ―――― 这就要用到递归。
